This is standard representation theory: SO(3) acts on M_n(R) by conjugation, decomposing into irreps l = 0, 1, ..., n-1 with dims 1, 3, 5, ..., 2n-1.

### 3. Crossed Casimir Formula [CONJECTURE with structure]

The SO(3) Casimirs C_2(l) = l(l+1) are themselves framework quantities:
- C_2(0) = 0
- C_2(1) = 2 = C
- C_2(2) = 6 = C * Im_H

The non-trivial weights satisfy: **w_l = (-1)^(l+1) * C_2(Im_H - l)**
- l=1: w = (-1)^2 * C_2(2) = +6 = C*Im_H
- l=2: w = (-1)^3 * C_2(1) = -2 = -C

Each irrep's weight is (up to sign) the Casimir of the OTHER non-trivial irrep. This is a "crossed coupling" structure.

Their coupling to EM channels goes through a different mechanism (anticommutator rather than commutator), introducing a sign flip. This allows destructive interference.

The sum 18 + (-10) + 1 = 9 = Im_H^2 is a unitarity/completeness condition: total signed weight across all processes equals total mode count.
