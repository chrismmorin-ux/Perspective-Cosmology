# DEF_0207 Definition: Orthonormal Basis

**Tag**: 0207
**Type**: DEFINITION
**Status**: CANONICAL
**Source**: core/01_universe.md

---

## Requires

- [DEF_0200: Notation conventions]
- [DEF_0205: Value space V]

## Provides

- B: Orthonormal basis of V

---

## Statement

**B** (Orthonormal Basis)

```
B = {b_1, ..., b_n} ⊂ V
```

Where:
- ⟨b_i, b_j⟩ = δ_ij (orthonormality)
- span(B) = V (completeness)
- n = dim(V)

---

## Notes

The basis B provides a canonical decomposition for content values.
See LEM_0402 for the unique decomposition property.
