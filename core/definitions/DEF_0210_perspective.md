# DEF_0210 Definition: Perspective

**Tag**: 0210
**Type**: DEFINITION
**Status**: CANONICAL
**Source**: core/02_perspective.md

---

## Requires

- [DEF_0200: Notation conventions]
- [DEF_0201: Universe tuple U]

## Provides

- π: The perspective triple

---

## Statement

A **perspective** is a triple:

```
π = (p, D, A)
```

Where:
- p ∈ P: Anchor point (see DEF_0211)
- D ⊂ T_p: Direction set (see DEF_0212)
- A: U → U_π: Access map (see DEF_0213)

---

## Notes

The perspective is the central primitive of this framework.
It represents "a point of view" on the universe U.
