# DEF_0213 Definition: Access Map

**Tag**: 0213
**Type**: DEFINITION
**Status**: CANONICAL
**Source**: core/02_perspective.md

---

## Requires

- [DEF_0201: Universe tuple U]
- [DEF_0210: Perspective π]

## Provides

- A: Access map from U to U_π

---

## Statement

**A: U → U_π** (Access Map)

Maps global content to accessible content from the perspective.

The access map determines what portion of the universe content
is visible/accessible from a given perspective.

---

## Notes

The access map is derived from propagation (see propagation module).
Properties constrained by axioms AXM_0104, AXM_0105, AXM_0106.
