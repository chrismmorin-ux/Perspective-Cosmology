- B-structure formalization

---

## Statement

**B** is the orthonormal basis of V:

```
B = {b_1, b_2, ..., b_n}
⟨b_i, b_j⟩ = δ_ij
dim(V) = n = |B|
```

---

## Notes

The B-structure provides the canonical decomposition for all content values.
