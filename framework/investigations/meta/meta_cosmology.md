### Definition: Unorthogonal Dimensions

**Definition (Unorthogonal Dimension Set)**

For a perspective π, the **unorthogonal dimension set** is:


These are the dimension pairs that deviate from perfect orthogonality under π.

The **total unorthogonality** of a perspective is:

```
U(π) = ||ε_ij||_F = √(Σ_{i≠j} |ε_ij|²)   (Frobenius norm)
```

**Interpretation**:
- U(π) = 0 → Perspective sees only orthogonal structure (no defect contribution)
- U(π) > 0 → Perspective carries unorthogonal structure
