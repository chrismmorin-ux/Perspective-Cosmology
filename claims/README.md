## How Random Matching Works

### The Flexibility Test

Given framework numbers {1, 2, 3, 4, 7, 8, 11, 12, 13, 14, 15, 17, 19, 21, 33, 37, 43, 44, 53, 72, 73, 77, 97, 99, 111, 121, 133, 137, 154...}

And formula templates: `alpha^n x p/q` or `alpha^n x pi x p/q`

How many random numbers in [0.1, 10] can be matched?

### Results

| Tolerance | Random Match Rate |
|-----------|-------------------|
| 5% | 100% |
| 1% | 100% |
| 0.1% | 86% |
| 0.01% | 31% |
| 0.001% | 0% |

### Interpretation

**For individual claims**: Matching a constant at percent-level proves little by itself.

**For the framework as a whole**: The question is whether 45+ coherent predictions from the SAME algebraic source is itself unlikely. The framework either describes real physics or is an unusually comprehensive coincidence.

Sub-10 ppm matches are individually significant. Percent-level matches gain significance through coherence.
