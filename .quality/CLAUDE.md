# Quality Engine Directory

Automated quality scanning infrastructure for the Perspective Cosmology repo.

## Files

| File | Purpose | Updated By |
|------|---------|------------|
| `report.md` | Latest scan results (overwritten each run) | `/quality-engine` |
| `history.md` | Append-only log of run summaries | `/quality-engine` |

## Usage

Run `/quality-engine` to scan the repo for structural issues, content gaps, consistency problems, and to score investigation priorities.

Reports compare against previous runs to track improvement over time.
